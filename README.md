# AIR PIANO 🎹

用手勢喺空中彈琴!利用 MediaPipe 手部追蹤 + Web Audio API,唔使任何實體琴鍵,對住鏡頭郁手指就可以彈出旋律。

🔗 **[立即試玩](https://tomleung11123.github.io/air-piano/)**

## 功能

- 即時手部追蹤,辨識指法郁動
- 內置歌曲:晴天、路小雨、說謊者、浮誇
- 純前端運作,唔使安裝任何嘢,瀏覽器開到就玩到
